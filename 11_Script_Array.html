배열

자바스크립트(js)에서는 변수 선언 시 별도의 자료형 지정 없이 어떤 자료형이든 배열에 담을 수 있다. 따라서 크기(길이)의 제약도 없다. (자바 컬렉션과 유사)

배열의 선언

콘솔창으로 내용 확인


* 배열 관련 메소드(함수)

1) 배열명.indexOf(찾고자하는값) : 배열에서 해당 값이 위치한 인덱스 값

2) 배열명.concat(배열, 배열, ...) : 배열에 여러 개의 배열을 결합하고자 할 때 사용


3) 배열명.reverse() : 배열에 담긴 값들을 역순으로 바꿔주는 함수


4) 배열명.sort() : 배열 내의 값을 오름차순 정렬시키는 함수



5_1) 배열에 값을 추가 : 배열명.push(추가할 값) : 배열의 맨 뒤에 값을 추가하고, 배열의 크기를 반환

5_2) 배열에 값을 제거 : 배열명.pop() : 배열의 맨 뒤에 값을 제거하고, 그 값을 반환


6_1) 배열명.unshift(추가할 값) : 배열의 맨 앞에 값을 추가하고, 배열의 크기를 반환

6_2) 배열명.shift() : 배열의 맨 앞에 값을 제거하고, 그 값을 반환


7_1) 배열명.slice(시작인덱스, 끝인덱스) : 배열로부터 시작인덱스 위치에서 끝인덱스 전까지 배열의 값들을 추출해주는 함수

7_2) 배열명.splice(시작인덱스, 제거할개수[, 추가할 값]) : 배열의 아이템을 추출해서 제거 및 추가해주는 함수


8) 배열명.toString() / 배열명.join([구분자]) : 배열의 각 인덱스에 담긴 값들을 하나의 "문자열"로 합쳐서 반환해주는 함수
