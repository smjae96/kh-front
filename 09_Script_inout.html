2. 데이터를 입력받는 구문 (변수에 저장 가능)

3) 변수 = 선택한요소.속성(id, className, innerHTML, innerText, ..)

1) [window.]confirm("질문 내용");

confirm 사용 시 "질문 내용"과 "확인/취소" 버튼이 존재하는 알림창이 화면에 표시

확인버튼 클릭 시 true, 취소버튼 클릭 시 false를 반환

2) [window.]prompt("질문 내용");

prompt 사용 시 "질문 내용"과 입력할 수 있는 "텍스트 상자", "확인/취소" 버튼이 존재하는 알림창이 화면에 표시 확인버튼 클릭 시 텍스트 상자에 입력된 내용을 문자열로 반환 취소버튼 클릭 시 null로 반환
